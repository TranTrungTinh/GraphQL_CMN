# Basic Demo GraphQL
  Example project using GraphQL

## Installation
```
git clone https://github.com/TranTrungTinh/GraphQL_CMN.git
```

```
cd GraphQL_CMN
``` 

```
npm/yarn install
``` 

## Usage
```
npm/yarn start http://localhost:4000/grapql
```

```
npm/yarn run json:server http://localhost:3000
```
